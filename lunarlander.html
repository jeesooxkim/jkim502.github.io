Python Game Using PyGame

As a team, two friends and I recreated the Lunar Lander arcade game using Python. This game uses principles of physics to guide a small spacecraft, aka a "lunar lander", safely to the surface of a planet of choice.

My role was to implement Pygame and get the keyboard input to function properly, as well as design the graphics so that they were as similar as possible to the original game. This was my first time implementing a the PyGame modules.
