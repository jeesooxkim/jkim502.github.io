Institute of Cellular Engineering

Cahan Lab focuses on accurate and efficient analysis of RNA-seq data through developing a cloud-based platform (written in R language) called CellNet in order to improve the accuracy of stem cell differentiation with respect to their target cells. This platform utilizes Amazon Web Services.

My current project focuses on human RNA-seq data collection of biopsied samples from mass online databases, such as GEO accession data and GTX data, and improving the efficiency of the R- and Python-based CellNet pipeline in order for quicker processing and analysis of existing human RNA-seq accessions. I debugged the program and made changes so that human RNA-seq data is also able to be processed. ! am currently feeding collected data through analysis algorithms, and obtaining standard data which will eventually serve as a baseline for accuracy comparison for ex vivo differentiated cells.
